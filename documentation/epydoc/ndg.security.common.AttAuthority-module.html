Package AttAuthority

source code

NDG Security Attribute Authority client - client interface classes to the Attribute Authority. These have been separated from their original location in the SecurityClient since they have the unusual place of being required by both client and server NDG security packages. For the server side they are required as the CredWallet invoked by the Session Manager acts as a client to Attribute Authorities when negotiating the required Attribute Certificate.

Make requests for Attribute Certificates used for authorisation
