MyProxy Client interface

Developed for the NERC DataGrid Project: http://ndg.nerc.ac.uk/

Major re-write of an original class. This updated version implements methods with SSL calls with M2Crypto rather use calls to myproxy client executables as in the original. This version is adapted and extended from an original program myproxy_logon by Tom Uram <turam@mcs.anl.gov>
