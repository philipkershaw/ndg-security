Class AAUserRoles

source code

An abstract base class to define the user roles interface to an Attribute Authority.

Each NDG data centre should implement a derived class which implements the way user roles are provided to its representative Attribute Authority.

Roles are expected to indexed by user Distinguished Name (DN). They could be stored in a database or file.
