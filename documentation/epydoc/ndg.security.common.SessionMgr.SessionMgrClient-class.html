getAttCert(self, userCert=None, sessID=None, attAuthorityURI=None, attAuthorityCert=None, reqRole=None, mapFromTrustedHosts=True, rtnExtAttCertList=False, extAttCertList=[], extTrustedHostList=[])

source code 

Request NDG Session Manager Web Service to retrieve an Attribute Certificate from the given Attribute Authority and cache it in the user's credential wallet held by the session manager.

ac = getAttCert([sessID=i]|[userCert=p][key=arg, ...])
Parameters:
  • userCert (string) - user certificate - use as ID instead of session ID. This can be omitted if the message is signed with a user certificate. In this case the user certificate is passed in the BinarySecurityToken of the WS-Security header
  • sessID (string) - session ID. Input this as an alternative to userCert in the case of a browser client.
  • attAuthorityURI (string) - URI for Attribute Authority WS.
  • attAuthorityCert (string) - The Session Manager uses the Public key of the Attribute Authority to encrypt requests to it.
  • reqRole (string) - The required role for access to a data set. This can be left out in which case the Attribute Authority just returns whatever Attribute Certificate it has for the user
  • mapFromTrustedHosts (bool) - Allow a mapped Attribute Certificate to be created from a user certificate from another trusted host.
  • rtnExtAttCertList (bool) - Set this flag True so that if the attribute request is denied, a list of potential attribute certificates for mapping may be returned.
  • extAttCertList (list) - A list of Attribute Certificates from other trusted hosts from which the target Attribute Authority can make a mapped certificate
  • extTrustedHostList (list) - A list of trusted hosts that can be used to get Attribute Certificates for making a mapped AC.
Returns: ndg.security.common.AttCert.AttCert
if successful, an attribute certificate.
Raises:
  • AttributeRequestDenied - this is raised if the request is denied because the user is not registered with the Attribute Authority. In this case, a list of candidate attribute certificates may be returned which could be used to retry with a request for a mapped AC. These are assigned to the raised exception's extAttCertList attribute
