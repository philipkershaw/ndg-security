OldchkCAPassphrase(self, caPassphrase=None)

source code 

Check given pass-phrase is correct for CA private key

This method allows checking of the pass-phrase without having to call sign() to sign a new certificate. It makes use of an openssl call where the pass-phrase is required - creation of a CRL (Certificate Revokation List)

source code 
Sign a certificate request
Parameters:
  • caPassphrase (string) - pass-phrase for SimpleCA's private key.
  • passphraseFilePath (string) - alternative to caPassphrase, Set pass-phrase from a file.
  • CN (string) - common name component of Distinguished Name for new cert. This keyword is ignored if certReq keyword is set.
  • certReq (M2Crypto.X509.Request) - certificate request
  • **createCertReqKw (dict) - keywords to call to _createCertReq - only applies if certReq is not set.
Returns: tuple
signed certificate and private key. Private key will be None if certReq keyword was passed in
