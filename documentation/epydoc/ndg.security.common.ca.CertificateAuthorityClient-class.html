Send a certificate request to the CA for signing

signCert([certReq=cr]|[CN=cn, openSSLConfigFilePath=p, **kw])
Parameters:
  • certReq (M2Crypto.X509.Request) - X.509 certificate request. If omitted, _createCertReq method is called to create a new public and private key and a certificate request
  • CN (string) - common name component of Distinguished Name for new cert. This keyword is ignored if certReq keyword is set.
  • openSSLConfigFilePath (string) - file path for OpenSSL configuration file from which to get settings for Distinguished Name for new certificate. This keyword is ignored if certReq keyword is set.
  • **createCertReqKw (dict) - keywords to call to _createCertReq - only applies if certReq is not set.
Returns: tuple
