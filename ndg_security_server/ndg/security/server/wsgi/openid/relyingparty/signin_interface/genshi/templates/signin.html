OpenID is a technology which enables you to sign in at different websites using the same identity. Rather than having to remember usernames/passwords for all the different sites you register with, you can use a single identity URL called an OpenID which is recognised by them all. The OpenID identifies you and your home site where you have an account and can login. At a site like this one, you can type in your OpenID, in the text box provided. When you select "Go", your browser will redirect you to your home site (your OpenID Provider) so that you can login. Once you've logged in, your home site will redirect you back to this site. This site will recognise you as signed in so that you access secured resources.

The default setting in the text box is the OpenID corresponding to this site. If you have an account registered here you can simply select "Go" and you will be redirected to the login page. You can also log in using an OpenID from another site but please note that this site may restrict sign in to a list of trusted OpenID Providers. If you have trouble using an OpenID with this site please contact the site administrator. If you don't know what you OpenID is please get in touch with your home site and find out if they can provide you with an OpenID.
