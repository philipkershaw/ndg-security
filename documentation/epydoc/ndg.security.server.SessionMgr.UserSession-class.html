encodeSessionMgrURI(txt, encrKey=None)
Static Method

source code 

Encode Session Manager URI to allow inclusion in a web browser session cookie

The address is optionally encrypted and then base 64 encoded use a URL safe encoding

Parameters:
  • encrKey (string) - 16 char encryption key used to encrypt the URI. If omitted or set None, the URI is not encrypted but merely base 64 encoded

decodeSessionMgrURI(txt, encrKey=None)
Static Method

source code 

Decode the URI from cookie set by another Session Manager. This is required when reading a session cookie to find out which Session Manager holds the client's session

Parameters:
  • txt (string) - base 64 encoded encrypted text
  • encrKey (string) - 16 char encryption key used to encrypt the URI. If omitted or set None, the URI is assumed to be unencrypted

createCookie(self, sessMgrURI, encrKey, sessID=None, cookieDomain=None, asString=True)

source code 

Create cookies for session ID Session Manager WSDL address

Parameters:
  • sessMgrURI (string) - address for Session Mananger
  • encrKey (string) - encryption key used to encrypted above URIs
  • sessID (string) - if no session ID is provided, use the latest one to be allocated.
  • cookieDomain (string) - domain set for cookie, if non set, web server domain name is used. Nb. Generalised domains which don't set a specific host can be a security risk.
  • asString (bool) - Set to True to return the cookie as string text. If False, it is returned as a SessionCookie instance.
Returns: SessionCookie / string depending on asString keyword
